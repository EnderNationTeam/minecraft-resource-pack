> 🛠️ Das Offizielle **EnderNation-Resource-Pack**!
> 🌌 Dieses Pack wurde exklusiv für den EnderNation-Server entwickelt, um euer Spielerlebnis auf ein völlig neues Level zu heben.

---

## 📂 Ordnerstruktur

- 📁 **`bedrock/`** – Texturenpack für **Minecraft Bedrock Edition**
- 📁 **`geysermc/`** – Mappings und Anpassungen für **GeyserMC**
- 📁 **`images/`** – Sonstige Bilder und Vorschau-Screenshots
- 📁 **`java/`** – Texturenpack für **Minecraft Java Edition**

### 📂 Installation (Java Edition)

1. Datei **nicht entpacken**
2. In den Resourcenpack-Ordner im Minecraft- Verzeichnis verschieben \
   Der Standart part ist `%appdata%\.minecraft\resourcepacks`
3. In Minecraft unter **Optionen → Ressourcenpakete** aktivieren ✅

### ⚠️ Hinweis

Dieses Resource-Pack wurde speziell für die Nutzung auf dem EnderNation-Server entwickelt und optimiert. Es könnte auf anderen Servern nicht ordnungsgemäß funktionieren oder die gewünschten Effekte erzielen.
